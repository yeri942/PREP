# PREP

가수 PREP의 팬 사이트입니다.
<br>

## 프로젝트 소개

> 전세계 힙스터에게 가장 많은 사랑을 받고 있는 아티스트라 해도 과언이 아닐 4인조 브리티쉬 밴드 프렙(PREP). 그들에 대해 소개합니다.

## 프로젝트 기획 의도

> 영국의 밴드 “PREP”의 팬으로써 “PREP”을 소개하는 사이트를 만들고 싶어 기획하게 되었습니다.
